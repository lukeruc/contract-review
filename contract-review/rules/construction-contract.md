# 建设工程施工合同审核规则

> 本规则为示例文件，展示审核规则的标准格式和内容结构。实际规则应由专业律师按合同类型编写维护。

## 一、合同主体与资质

- [ ] 核查发包人是否为项目合法建设单位（土地使用权证、规划许可证等）
- [ ] 核查承包人是否具备相应施工资质等级，资质是否在有效期内
- [ ] 联合体承包的，各成员方资质是否满足要求，联合体协议是否明确各方责任
- [ ] 是否存在违法转包、违法分包的约定或暗示
- [ ] 承包人项目经理是否具备相应执业资格

## 二、工程范围与工期

- [ ] 工程范围的描述是否明确无歧义（工程量清单、图纸、技术规范是否齐备且一致）
- [ ] 开工日期、竣工日期是否明确
- [ ] 中间交工节点（如有）的日期和验收标准是否明确
- [ ] 工期延误的责任分配与违约金标准是否合理
- [ ] 发包人原因导致的工期顺延条件是否充分
- [ ] 不可抗力对工期影响的处理机制

## 三、合同价款与支付

- [ ] 合同价款类型是否明确（固定总价 / 可调价 / 成本加酬金）
- [ ] 固定总价的风险范围及风险范围以外合同价格的调整方法
- [ ] 预付款金额、支付时间、抵扣方式
- [ ] 进度款支付周期、支付比例、审批流程
- [ ] 竣工结算的程序、时限、争议解决方式
- [ ] 质量保证金比例（不高于工程价款结算总额的 3%）
- [ ] 变更洽商的价格调整机制是否完整
- [ ] 逾期付款的违约责任（利息标准）

## 四、工程质量与验收

- [ ] 工程质量标准是否明确（国标、行标、合同约定标准）
- [ ] 隐蔽工程验收的程序和时限
- [ ] 中间验收与竣工验收的条件和程序
- [ ] 质量不合格的返工责任与费用承担
- [ ] 质量保修范围和保修期限（是否短于法定最低保修期）
- [ ] 保修期起算时间是否明确

## 五、安全生产与文明施工

- [ ] 安全生产责任的分配（是否符合《安全生产法》规定）
- [ ] 安全生产费用的提取与使用
- [ ] 安全事故的报告与处理程序
- [ ] 工伤保险和意外伤害保险的购买义务

## 六、违约责任

- [ ] 违约情形是否对等（发包人与承包人的违约责任是否平衡）
- [ ] 违约金比例是否合理（是否过分高于实际损失）
- [ ] 是否有违约金上限约定，上限是否合理
- [ ] 解除合同的条件是否明确、对等
- [ ] 合同解除后的结算与清场程序

## 七、争议解决

- [ ] 争议解决方式（诉讼/仲裁）是否明确
- [ ] 管辖法院或仲裁机构是否明确且对己方有利
- [ ] 是否约定了争议解决前的协商/调解前置程序
- [ ] 适用法律是否明确

## 八、其他重要条款

- [ ] 保险条款：工程一切险、第三者责任险的购买义务和受益人
- [ ] 知识产权条款：图纸、设计方案的知识产权归属
- [ ] 保密条款：保密范围和期限
- [ ] 转让与分包限制
- [ ] 合同生效条件是否明确
- [ ] 合同文件的组成与优先解释顺序
