# 工程总承包（EPC/交钥匙）合同审核规则

> 本规则从承包商立场出发，聚焦 EPC 合同特有的核心商务风险。

## 一、合同价格与支付

- [ ] 预付款金额是否已在合同数据中明确约定
- [ ] 保留金比例是否过高（超过 5%），累计上限是否合理
- [ ] 保留金释放后是否被要求以保函替代
- [ ] 付款币种中不可自由兑换的当地货币占比是否过高，汇率风险由谁承担
- [ ] 增值税/预提税/关税是否均由承包商承担

## 二、工期与延误

- [ ] "不可预见困难"和"不可预见地质条件"是否被明确排除在 EOT 事由之外
- [ ] 同期延误时 EOT 的计算规则是否在专用条件中有明确约定
- [ ] 误期损害赔偿每日费率是否过高，是否有总额上限
- [ ] 业主暂停超 84 天后承包商是否保留终止权

## 三、设计责任

- [ ] 承包商是否对业主需求文件的准确性、充分性和完整性承担全部责任
- [ ] 业主对设计文件的审查、无异议或批准是否明确不解除承包商的任何设计责任
- [ ] 业主审查设计文件的时限是否过短（默认 21 天）
- [ ] 设计标准与项目所在国强制性规范冲突时，协调成本由谁承担
- [ ] 设计成果的知识产权在业主便利终止后是否仍被业主使用

## 四、变更与价格调整

### 变更（Variations）

- [ ] 业主单方变更权是否无合理限制
- [ ] 承包商对不合理变更指令是否保留了拒绝权（不可预见、无法采购、影响安全环保、影响性能保证、影响意图用途）
- [ ] 变更价格未确定前业主是否须按暂定价先行支付

### 法律变更（Changes in Laws）

- [ ] 基准日期后法律变化导致成本增加，合同价格是否自动调增
- [ ] 法律变更的覆盖范围是否包含新法颁布、旧法修订、司法解释和政府许可要求变化
- [ ] 法律变更导致成本减少时业主是否可同比例下调
- [ ] 法律变更引起工期延误时承包商是否可同时获得 EOT

### 成本波动（Changes in Cost）

- [ ] 专用条件中是否附有成本指数调整表（未附则该条整体不生效）
- [ ] 调价公式覆盖的成本要素是否匹配承包商实际成本结构
- [ ] 各成本要素的权重和调价频率是否合理
- [ ] 多币种支付时各币种是否有独立的调价公式

## 五、履约保障与现场风险

- [ ] 履约担保额是否过高（超过合同价格 10%）
- [ ] 履约担保是否须维持至签发履约证书且清场完毕
- [ ] 业主兑现履约担保的情形是否严格限于合同明示
- [ ] 业主超额兑现履约担保是否须全额赔偿承包商
- [ ] 合同是否明确排除了"不可预见物理条件"的索赔权

## 六、缺陷责任

- [ ] 承包商未及时修复时，业主自行或委托修复的扣款是否有上限

## 七、例外事件与不可抗力

- [ ] 承包商可获的救济是否仅限于 EOT（战争和自然灾害修复费除外）
- [ ] 例外事件持续 84/140 天后承包商是否有权终止合同
- [ ] 工程照管责任是否随接收证书签发而转移至业主

## 八、终止合同

- [ ] 业主以承包商违约为由终止前是否必须先发出纠正通知
- [ ] 承包商的终止事由是否穷尽列明、无兜底条款
- [ ] 业主便利终止的赔偿是否明确包含承包商的利润损失
- [ ] 便利终止后履约担保是否应立即退还
- [ ] 承包商暂停权（21 天通知）和终止权（付款逾期 42 天等）的门槛是否被专用条件抬高

## 九、索赔程序

- [ ] 索赔通知时限是否为 28 天，超期是否完全失权
- [ ] 详细索赔书的提交时限是否为 84 天
- [ ] 同期记录的制作和保存是否已成为承包商的合同义务
- [ ] 业主的索赔是否受完全相同的时限约束
- [ ] 前述时限在适用中国法的合同中是否可能被认定为无效格式条款

## 十、争议解决

- [ ] 仲裁机构、规则、地点、语言是否明确，仲裁地是否为中立法域
- [ ] 项目所在国是否为《纽约公约》缔约国
