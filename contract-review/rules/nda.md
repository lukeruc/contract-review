# 保密协议审核规则

## 一、保密信息的定义

- [ ] 是否有专门的保密信息定义条款
- [ ] 定义是否具体列举了保密信息的类型和范围（如技术资料、客户名单、财务数据等），而非仅作概括性描述

## 二、兜底条款

- [ ] 保密信息定义中是否存在"包括但不限于"、"以及其他一切"、"或其他类似信息"等兜底表述。如有，应建议删除
- [ ] 是否存在"披露方单方认定为保密信息"的表述。如有，应建议修改——保密范围不应由一方单方决定
- [ ] 是否存在"所有披露的信息均视为保密信息"的概括性表述。如有，应建议修改
- [ ] 是否设置了保密信息标识要求（如标注"机密"），使接收方能客观识别保密信息

## 三、保密期限

- [ ] 是否明确约定了保密期限
- [ ] 保密期限是否合理（不应为"永久"，通常 2-5 年）
- [ ] 是否区分了协议存续期和协议终止后的保密义务存续期间
- [ ] 协议终止或应披露方要求时，接收方是否负有返还或销毁保密信息的义务

## 四、双方权利义务

- [ ] 接收方的使用范围是否限制为"仅为约定目的使用"
- [ ] 接收方可向哪些人员披露是否明确（如"仅限有必要知悉的员工"）
- [ ] 是否约定了返还或销毁义务
- [ ] 强制披露（司法程序、监管要求）时的处理方式是否约定
- [ ] 是否涉及到除了保密以外的权利义务
